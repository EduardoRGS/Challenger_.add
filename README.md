# Challenger_.add
CRUD de um Aplicação de gerenciamento escolar

## ✔️ Tecnologias utilizadas:
-``Java 11``
-``SpringBoot``
-``SpringBoot JPA``
-``SpringBoot WEB``
-``SpringBoot DevTools``
-``Thymeleaf``
-``Postresql``
-``Swagger (Springfox)``
-``Lombok``

## Para rodar a api:
### Baixe todas as dependências do meven que estão no pom.xml

## Para alterar o banco de dados:
